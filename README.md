# INSTAKIT
Instakit is a tool used to extract information from other people's/yourself Instagram accounts without logging in/blocking

# Results
* Username
* ID
* Full name
* Biography
* Business category name
* External URL
* Followed back
* Following
* Follower
* Following back
* Blocked people
* Never blocked people
* Has a highlight
* Have a public story
* Have asked people
* Asked people
* Has a story to watch
* IGTV
* Business account
* Private account
* Verified
* Post
* Profile photo URL
#Tested on
Windows

- Instaloader
